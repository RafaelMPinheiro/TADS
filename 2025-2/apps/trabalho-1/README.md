# Trabalho I - Quiz
## Conteúdos Obrigatórios:

- Controle de State
- Controle de props de componentes
- Controle de navegação através de views
## Motivação do Trabalho:

- Entender e utilizar o React Native junto com os conceitos principais e básicos relacionados a controle de state, a componentização de uma aplicação front-end e navegação entre múltiplas telas.
## Entregável

**Aplicativo com 3 telas:**

1. **Tela de Cadastro:** Simular um cadastro, informando o nome e email, estes campos serão passados para as próximas telas.

2. **Tela do Quiz:** Segunda tela mostrando uma pergunta de cada vez, com pelo menos 5 questões de múltipla escolha (3+ alternativas).

3. **Tela de Resultado:** A última tela com a mensagem de que o quiz foi completado, mostrando o nome e email do usuário e a nota final.
## Observações

- A temática do Quiz é livre.
- Frontend utilizável.
